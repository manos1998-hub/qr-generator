How to Use:

  1. Enter your app URL - Paste the link to your working restaurant app
  2. Select a table - Choose from the dropdown or enter a custom table name
  3. Generate QR Code - Click "Generate QR Code" to create the QR code
  4. Download & Print - Download the QR code and print it for the table
  5. Repeat for all tables - Generate QR codes for each table in your restaurant

Tip: Print and laminate the QR codes for durability. Place one on each table.
